Method: verifyPhoneNumber

Request Payload

Parameter

Value

name

 Name of the user being registered

phone

 Phone number to be registered

onCompletion

 Completion handler to call back when the authorization process completes

Response Payload

Status

Value

Success

 onCompletion block is called with nil parameter

Failure

 onCompletion block is called with error

Note

  • Creates a new application instance/account, updates the phone number or any other attributes of an existing application instance/account.

  • On successful verification of the phone number, the OTP entry screen is displayed.

  • On successful verification of the phone number, the server sends an OTP to the phone number.

  • On failure to verify the phone number, an error message is sent to the server.

Method: verifyOTP

Request Payload

Parameter

Value

 Phone number to be D1registered

OTP

6 digit OTP send to the phone number

onCompletion

Completion handler to call back when the verifaction OTP process completes

Response Payload

Status

Value

Success

onCompletion block is called with 'nil' parameter

Failure

onCompletion block is called with error

Note

  • Verifies the 6 digit OTP to authenticate the user.

  • On successful verification of OTP, the 'nil' parameter is sent.

  • On failure to verify the OTP, an error message is sent.
