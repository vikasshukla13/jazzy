Method: fetchContacts

Request Payload

onCompletion: Completion handler to call back when the fetch contacts process completes.

Response Payload

Status

Value

Success

onCompletion block is called with

data

The list of contacts

error

Nil

Failure

onCompletion block is called with

data

Nil

error

An error object

Note

  • Retrieves the entire telephone numbers list from the enterprise telephone numbers list.

  • On successful retrieval of phone numbers, a list of contacts is sent.

  • On failure to retrieve phone numbers, an error is sent.
