Method: registerForRemoteNotification

Request Payload

token: The device token received by application delegate.

Response Payload

Irrespective of whether the method fails or executes successfully, there is no response.

Note

  • Sends the device token to server, saves the device token locally.

  • The server uses the device token to send push notification when the device receives a inbound call.
